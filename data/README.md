# greek_ephesians_2.json

Original-language (Koine Greek) word data for Ephesians 2 -- Strong's
number, the real inflected Greek word as it appears in the text (in
correct Greek word order), a deterministic transliteration, and a short
lexical definition per word.

**Source**: derived from [tahmmee/interlinear_bibledata](https://github.com/tahmmee/interlinear_bibledata)
(public domain), itself built from the public-domain Strong's Concordance
and a KJV-aligned interlinear New Testament. See
`scripts/build_greek_data.py` for exactly how this file was generated.

**Why not generate this with Gloo instead?** LLMs are known to hallucinate
plausible-but-wrong Strong's numbers, Greek roots, and transliterations --
this is exactly the kind of fact that must come from a real reference
source, not be generated. Gloo is only used to *explain* a real, looked-up
entry, never to invent the entry itself.

**Scope note**: this data aligns to the Greek New Testament's own word
order, not to any specific modern translation's word order (English,
Chinese, Japanese, etc. all phrase things differently). So it's presented
as "the original words in this verse," not as a precise 1:1 mapping to
whichever word you tapped in the primary reading pane.
